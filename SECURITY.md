# Security Policy

## Supported disclosure

If you find a security issue in AgentBus, please report it privately before publishing details.

## What to report

- secret leakage in task, result, memory, or inbox files
- permission mistakes in Android/Termux or local worker flows
- unsafe default actions in routing or worker execution
- workflow issues that could cause unintended publishing or looping

## What to include

- a short description of the issue
- the file path or workflow involved
- steps to reproduce
- any suggested fix or mitigation

## Response

We will review the report, confirm impact, and coordinate a fix before public disclosure.
